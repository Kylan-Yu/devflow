# DevFlow Admin

React + TypeScript + Vite admin app for moderation workflows.  
基于 React + TypeScript + Vite 的管理后台，用于承载治理和审核流程。

## Current Scope | 当前范围
- Admin login page (`/login`)
- Dashboard page (`/dashboard`)
- User moderation list (enable / disable community users)
- Post moderation list (hide / restore posts)
- Report moderation queue (review, dismiss, hide post, disable user)
- Recent admin audit log list
- Bilingual i18n (`en-US` / `zh-CN`)
- Language switcher
